## 📥 HƯỚNG DẪN CÀI ĐẶT

1. Tải mã nguồn extension về máy
2. Mở Chrome / Edge / Brave

4. Bật **Developer mode**
5. Nhấn **"Load unpacked"** → Chọn thư mục chứa extension
6. Extension sẽ tự động hoạt động trên YouTube
